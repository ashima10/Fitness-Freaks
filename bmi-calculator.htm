How is BMI calculated?            
                 
Formula to calculate BMI using metric units of measure.      
                 
BMI = weight(kg) / (height(cm) x height(cm))          
                 
Formula to calculate BMI using imperial units of measure.      
                 
BMI = (weight(lbs) / (height(inch) x height(inch))) x 703
